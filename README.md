# 🚀 Multi-Step Form with Live Preview & Local Storage

A modern **Multi-Step Form Web App** built using **HTML, CSS, and JavaScript**.  
It allows users to enter data step-by-step, preview it live, and store it in the browser using Local Storage.

🔗 **Live Demo:** https://multi-step-form-jet-one.vercel.app/

## ✨ Features

- 🧩 Multi-step form (Personal Info → Account → Skills)
- 👀 Live preview while typing
- 🖼️ Image upload with instant preview
- ➕ Dynamic experience field (add/remove)
- 💾 Data saved in Local Storage
- 📂 Sidebar to view submitted data
- ✏️ Edit & Delete functionality
- 🔔 Custom toast notifications
- 📱 Fully responsive UI

## ⚙️ How It Works

1. User fills data step-by-step
2. Live preview updates in real-time
3. On submit:
   - Data stored in Local Storage
   - Displayed in sidebar
4. User can:
   - Edit data
   - Delete data
5. Data persists after refresh
